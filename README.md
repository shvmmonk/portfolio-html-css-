## 📌 About

This is a single-page portfolio site highlighting my work as a CS student, including my projects, technical skills, and hackathon participation.

## 🛠️ Built With

- **HTML5** — semantic page structure
- **CSS3** — Flexbox, Grid, custom properties (CSS variables), transitions

No frameworks or libraries — built from scratch to strengthen core HTML/CSS fundamentals.

## 📂 Sections

- **Hero** — Introduction and tagline
- **About** — Brief background
- **Skills** — Languages, frameworks, and concepts, grouped by category
- **Projects** — Featured projects:
  - **Fake News Detector** — ML-based news classification system
  - **AI Integrated Drone** — Autonomous drone with AI-based object detection
- **Hackathons & Achievements** — Hackathon participation and builds
- **Contact** — Ways to get in touch

## 🎨 Design Notes

- Dark theme with a consistent color palette managed through CSS custom properties (`:root` variables)
- Responsive grid layouts for Skills, Projects, and Hackathons sections using `auto-fit` and `minmax()`
- Subtle hover animations on project cards for interactivity
